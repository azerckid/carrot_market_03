# 프로필 페이지 개선 기획안

## 📋 현재 상태

현재 프로필 페이지는 매우 기본적인 형태입니다:
- 사용자 이름만 표시
- 로그아웃 버튼만 있음
- 스타일링이 거의 없음

---

## 🎨 디자인 컨셉

### 전체 레이아웃
- **다크 테마** 유지 (다른 페이지와 일관성)
- **카드 형태의 섹션**으로 구분
- **스크롤 가능한 레이아웃** (하단 탭바 고려)

---

## 📐 화면 구성

### 1. 프로필 헤더 섹션
```
┌─────────────────────────────────┐
│  [큰 아바타 이미지]              │
│                                  │
│  사용자명                        │
│  이메일 (있는 경우)              │
│  가입일: X일 전                  │
└─────────────────────────────────┘
```

**구현 요소**:
- 큰 아바타 이미지 (원형, 중앙 정렬)
- 사용자명 (큰 글씨, 굵게)
- 이메일 (있는 경우만 표시, 작은 글씨)
- 가입일 (상대 시간 표시)

---

### 2. 통계 섹션
```
┌─────────────────────────────────┐
│  내 활동                        │
│  ┌─────┐ ┌─────┐ ┌─────┐      │
│  │게시글│ │상품 │ │댓글 │      │
│  │  X  │ │  Y  │ │  Z  │      │
│  └─────┘ └─────┘ └─────┘      │
└─────────────────────────────────┘
```

**구현 요소**:
- 내가 작성한 게시글 수
- 내가 등록한 상품 수
- 내가 작성한 댓글 수
- 각 항목 클릭 시 해당 목록으로 이동 (선택사항)

---

### 3. 내 게시글 섹션
```
┌─────────────────────────────────┐
│  내 게시글                      │
│  ─────────────────────────────  │
│  [게시글 1]                     │
│  제목, 작성일, 조회수, 댓글수   │
│  ─────────────────────────────  │
│  [게시글 2]                     │
│  ...                            │
│  ─────────────────────────────  │
│  [더보기] → /life?user=me       │
└─────────────────────────────────┘
```

**구현 요소**:
- 최근 작성한 게시글 3-5개 표시
- 각 게시글 클릭 시 상세 페이지로 이동
- "더보기" 링크로 전체 게시글 보기

---

### 4. 내 상품 섹션
```
┌─────────────────────────────────┐
│  내 상품                        │
│  ─────────────────────────────  │
│  [상품 1] [상품 2] [상품 3]     │
│  (그리드 레이아웃)              │
│  ─────────────────────────────  │
│  [더보기] → /home?user=me       │
└─────────────────────────────────┘
```

**구현 요소**:
- 최근 등록한 상품 3-6개 표시 (그리드 레이아웃)
- 각 상품 클릭 시 상세 페이지로 이동
- "더보기" 링크로 전체 상품 보기

---

### 5. 설정 섹션
```
┌─────────────────────────────────┐
│  설정                           │
│  ─────────────────────────────  │
│  [로그아웃]                     │
└─────────────────────────────────┘
```

**구현 요소**:
- 로그아웃 버튼
- 향후 확장 가능 (프로필 수정, 알림 설정 등)

## 🎯 구현 우선순위

### Phase 1: 기본 프로필 정보 (필수)
1. 프로필 헤더 섹션
   - 아바타 이미지 표시
   - 사용자명, 이메일, 가입일 표시

2. 통계 섹션
   - 게시글 수, 상품 수, 댓글 수 조회 및 표시

3. 설정 섹션
   - 로그아웃 버튼 스타일 개선

**예상 소요 시간**: 1-2시간

---

### Phase 2: 내 게시글/상품 목록 (추가)
4. 내 게시글 섹션
   - 최근 게시글 목록 표시
   - 게시글 상세 페이지로 링크

5. 내 상품 섹션
   - 최근 상품 그리드 표시
   - 상품 상세 페이지로 링크

**예상 소요 시간**: 2-3시간

### Phase 1 구현 내용

#### 1. 프로필 헤더 섹션
**파일**: `app/(tabs)/profile/page.tsx`

**데이터 조회**:
```typescript
const user = await db.user.findUnique({
  where: { id: session.id },
  select: {
    id: true,
    username: true,
    email: true,
    avatar: true,
    created_at: true,
    _count: {
      select: {
        posts: true,
        products: true,
        Comment: true,
      },
    },
  },
});
```

**UI 구성**:
- 큰 아바타 이미지 (중앙 정렬)
- 사용자명 (큰 글씨)

- 가입일 (formatToTimeAgo 사용)

---

#### 2. 통계 섹션
**UI 구성**:

- 호버 효과 (선택사항)

**아이콘**:
- 게시글: `NewspaperIcon`
- 상품: `ShoppingBagIcon` 또는 `CubeIcon`
- 댓글: `ChatBubbleBottomCenterIcon`

#### 3. 설정 섹션
**UI 구성**:

### Phase 2 구현 내용

#### 4. 내 게시글 섹션
**데이터 조회**:
```typescript
const myPosts = await db.post.findMany({
  where: { userId: session.id },

    id: true,

    description: true,
    views: true,
    created_at: true,
```

## 🎨 디자인 가이드

### 색상
- 배경: 다크 테마 (`bg-neutral-900` 또는 `bg-black`)
- 텍스트: 흰색 (`text-white`)
- 강조: 오렌지색 (`text-orange-500`, `bg-orange-500`)
- 구분선: `border-neutral-700`

### 간격
- 섹션 간격: `mb-8` 또는 `mb-10`
- 카드 내부 패딩: `p-5` 또는 `p-6`
- 요소 간격: `gap-4` 또는 `gap-5`

### 타이포그래피
- 제목: `text-2xl font-bold`
- 부제목: `text-lg font-semibold`
- 본문: `text-sm` 또는 `text-base`
- 작은 텍스트: `text-xs`
